[ 데이터 전처리 ]
원하는 데이터는 자동차의 X, Y 위치
그렇다면 주어진 데이터에서 X, Y와 상관관계가 깊은 것이 무엇인지 판단할 필요가 있다



[ 모델 ]
생각해보니 0.01초 다음의 위치를 계산하는 것은 현재 위치, 현재 속도/가속도 물리량만 알면 되는 것 아닌가?
신경망 input으로 위치를 주지 말고, 이전 n step의 속도/각속도/방향만 주고 0
신경망 output으로는 X, Y, Z pos의 변화량을 출력하면 어떨까?
그리고 신경망의 output과 현재 위치를 input으로 받고 둘을 더하여 output을 내는 모듈을 최종 결과물로 만들어보자

[ 모델 2 ]
시간이 남으면 플레이어의 인풋 상황만을 이용해서도 해보자

[ 모델 3]
마지막으로는 데이터를 고르지 말고 그냥 모든 데이터를 때려박아서 실험해보자
